This project analyzes global COVID-19 data to explore case trends, compare countries, and visualize infection rates over time using Python and Plotly.

Key Features:
Data cleaning and preprocessing,
Global choropleth maps (cases & deaths),
Daily infection rate calculation,
Maximum infection rate per country,
Case study on Italy’s lockdown impact

NOTE:
Some interactive figures (Plotly) did not render properly in the final output environment. They work correctly when run locally (e.g., Jupyter Notebook).
